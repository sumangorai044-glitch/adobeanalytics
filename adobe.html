Adobe Analytics Rule Tester

A playground to see Pageviews, Link Clicks, and Button Clicks interact with the Data Layer.

1. Pageview Rule

In Adobe Launch, this rule is triggered by a Core - Library Loaded (Page Top) or Page Bottom event.

It automatically reads the global digitalData object and executes an s.t() call.

2. Link Click Rule

In Adobe Launch, this uses the Core - Click event mapping to anchor tags a. It captures destinations.

Fires an s.tl() call with link type set to 'e' (Exit) or 'd' (Download).

3. Button / Custom Interaction Rule

Triggers on specific on-page element selections using ID or CSS class selectors. Does not reload or change the URL path.

Fires an s.tl() call with link type set to 'o' (Other Custom Link).
